# Fruit-image-classification
This project is a deep learning fruit image classification model built using Python, TensorFlow, and Keras. It uses a Convolutional Neural Network (CNN) to recognize different types of fruits from images by learning important visual features such as color, shape, and texture. The dataset is organized into folders, where each folder represents a fruit category, making it easy for the model to learn from labeled images. The dataset is was collected from Kaggle. Before training, the images are resized and normalized to improve consistency and model performance. This project covers the main steps of a computer vision workflow, including loading the dataset, preprocessing images, building and training the CNN model, testing predictions on unseen images, and saving the trained model for future use. The trained model file is not included because of its size, but it can be recreated by running the provided training script.
 Dataset link: https://www.kaggle.com/datasets/sshikamaru/fruit-recognition?resource=download
